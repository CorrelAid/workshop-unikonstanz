Exkurs String Manipulation - Regex

  • regex: reguläre Ausdrücke beschreiben Muster in Sprache
  • wichtigste Elemente
    • . : wildcard (Joker)
    • \d: digit
    • +: vorausgegangenes Element einmal oder mehr
    • ?: vorausgegangens Element optional
    • {n,m} / {n}: vorausgeganenes Element kommt mind. n und höchstens m mal vor / kommt genau n mal vor
    • [abc]: definiert eine Menge
  • ?regex
  • zum Testen: Regex101 & RegexR

Exkurs String Manipulation - Beispiel

  • Postleitzahlen haben das Format D-xxxx(x), z.B. D-78464
  • in regex:
    • D-\d{5} matcht alle fünfstelligen PLZ inkl. "D-"
    • D-\d{4,5} matcht alle vier- bis fünfstelligen PLZ inkl "D-"
    • \d{5}: matcht nur die fünfstelligen Zahlen
  • in RegexR

Merging/Joins - Warum Merging / Joins?

Herausforderung

  • wir haben oft mehrere Datenquellen, die wir zusammenfügen müssen
  • Beispiel: Umfrage-Datensatz und Länder-Datensatz, z.B. European Social Survey und Daten von Eurostat -> wollen die beiden Datensätze zusammenfügen

Merging/Joins - dplyr joins

dplyr join-Funktionen

  • left_join(x, y) includes all observations in x, regardless of whether they match or not.
  • inner_join(x, y) only includes observations that match in both x and y.
  • full_join() includes all observations from x and y.
  • …

Ressourcen / Links

  • Vignette für dplyr Joins
  • Visual Representation of SQL Joins ("Outer Join" = full_join)

Exkurs Pipes - Ceci n'est pas un pipe

Pipes

Exkurs dplyr

Exkurs dplyr - was ist dplyr?

  • dplyr package
  • "It provides simple “verbs”, functions that correspond to the most common data manipulation tasks, to help you translate those thoughts into code."
  • unterstützt Piping mit %>% \(\rightarrow\) integrierter Workflow

Exkurs dplyr - dplyr Verben

  • filter() (and slice()): Zeilen auswählen
  • arrange(): Zeilen anordnen / sortieren
  • select() (and rename()): Variablen/Spalten auswählen und umbenennen
  • distinct(): unique Zeilen extrahieren (Dubletten löschen)
  • mutate(): neue Variablen/Spalten erstellen
  • summarise(): Werte zusammenfassen
  • group_by: Operationen nach Gruppen durchführen (z.B. summarise)
